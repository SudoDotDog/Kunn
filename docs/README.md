:dolphin: Interfacing, between people, between machine

## Install

```sh
npm install kunn -g
# OR
yarn global add kunn
```

## Development

```sh
make install
make link
make dev
# Linked environment path
...kunn {...}
```

## Test

```sh
make tests
```

## Build

```sh
make build
```
